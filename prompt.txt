Build a complete Online Resume Builder Web Application.

Tech Stack:
Frontend: HTML5, CSS3, Vanilla JavaScript (ES6)
Backend: Node.js + Express.js
Database: MongoDB Atlas (cloud)
Authentication: JWT + bcrypt
PDF Generation: jsPDF

Do NOT use React, Angular, Vue, or any frontend framework.

====================================
CORE REQUIREMENTS
====================================

1. Authentication System
- Register page (name, email, password)
- Login page
- Password hashing using bcrypt
- JWT token generation on login
- Protected routes using JWT middleware
- Logout functionality
- Proper frontend + backend validation
- Error handling messages

2. Dashboard
After login redirect to dashboard.
Dashboard should display:
- Create New Resume button
- List of Saved Resumes
- Edit Resume
- Delete Resume
- Download Resume as PDF

3. Resume Builder Form
Sections:
- Personal Details (name, email, phone, address, LinkedIn, GitHub)
- Professional Summary
- Education (dynamic add/remove entries)
- Experience (dynamic add/remove entries)
- Skills (comma separated or tag input)
- Projects
- Certifications
- Languages

4. Resume Templates
- At least 2 professional templates
- Template selection dropdown
- Live preview section
- Switching templates should not lose data

5. PDF Generation
- Use jsPDF
- Proper spacing and formatting
- Multi-page support
- File name format: username_resume.pdf

====================================
DATABASE STRUCTURE
====================================

User Schema:
- name
- email (unique)
- password
- createdAt

Resume Schema:
- userId (ObjectId reference to User)
- personalDetails
- summary
- education[]
- experience[]
- skills[]
- projects[]
- certifications[]
- languages[]
- templateType
- createdAt
- updatedAt
